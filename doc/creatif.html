Wiki : monde créatif

Introduction

Le monde créatif met à votre disposition une parcelle de 48 x 48 blocks. Vous pouvez y construire ce que vous désirez (dans la limite du raisonnable). Vous pouvez également acheter d'autres parcelles (vous pouvez en posséder jusqu'à 4). Vous pouvez visiter les parcelles des autres joueurs. Des outils de construction et de décorations sont mis à disposition tel que "World edit" et un accès aux bannières et aux têtes de joueurs plus rapide. Des évènements de constructions sont mis en place, avec en prime pour le(s) gagnant(s), l'exposition de leur(s) construction(s) dans le hub.

Gestion des parcelles

/plot buy

Achète le plot sur lequel vous vous trouvez (si il est à vendre).

/plot claim

Revendique le plot sur lequel vous êtes si il n’a pas de propriétaire. Lorsque vous revendiquez un plot, il vous appartient et personne ne peut construire dessus à part vous et ceux que vous autorisez.

/plot auto

Revendique automatiquement le plot le plus proche du spawn. Il est possible de spécifier une taille pour revendiquer plusieurs plots si besoin.

/plot delete

Supprime le plot et le rend de nouveau revendiquable (vous n’en serez plus le propriétaire). Cette action est irréversible. Cette commande demande une confirmation.

/plot setowner

Change le propriétaire du plot.

/plot setowner

Change le propriétaire du plot.

/plot download

Créer un fichier .schematic de votre plot (utilisable par WorldEdit, MCEdit, etc) et génère un lien pour le télécharger.

Téléportation

/plot visit

Permet de visiter un plot. On peux donner le pseudo d’un joueur, le nom d’un plot (alias) ou son id.

/plot kick

Ejecte un joueur du plot.

/plot middle

Vous téléporte au milieu du plot actuel.

Paramètres

/plot trust

Autorise un joueur à construire dans un plot même si le propriétaire n’est pas en ligne.

/plot add

Autorise un joueur à construire sur le plot uniquement lorsque le propriétaire est en ligne.

/plot deny

Banni un utilisateur d’un plot. Il ne pourras plus y accéder tant que vous ne le débannissez pas avec la commande /plot remove.

/plot remove

Retire un joueur du plot. Si c’est un joueur autorisé à construire, il ne sera plus autorisé à construire, et si c’est un joueur banni, il sera débanni.

/plot merge

Combine le plot sur lequel vous vous trouvez avec un plot adjacent. Vous devez posséder l’autre plot pour pouvoir faire ceci. Vous pouvez spécifier une direction (n,e,w,s ou all) et spécifier si vous souhaitez retirer la route en ajoutant true ou false.

/plot unlink

Sépare des plots qui ont été combinés.

/plot setflag

Défini les flags d’un plot. Pour la liste des flag, utilisez /plot flag list. Pour plus d’information sur les flags, voir Flags.

/plot done

Marque un plot comme terminé.

/plot continue

Permet de continuer un plot qui était marqué comme terminé. .

/plot setdescription

Défini la description du plot.

/plot sethome

Défini le point de téléportation du plot lorsqu’on s’y téléporte en utilisant /plot visit.

/plot toggle

Active / désactive des paramètres utilisateurs :

  • worldedit : WorldEdit
  • chat : Le chat de plot
  • titles : Le texte affiché à l’écran quand on rentre dans un plot
  • time : Le changement de temps quand on rentre dans un plot avec un temps spécifique paramétré.

Discussion

/plot chat

Active/désactive le chat de plot (chat visible uniquement sur le plot).

Apparence

/plot set

Défini une valeur d’un plot. Permet de définir le biome, le nom et les flags.

/plot music

Permet de choisir une musique qui sera jouée sur le plot.

/plot clear

Réinitialise le plot et le remet à son état de départ. Cette action est irréversible. Cette commande demande une confirmation.

/plot setbiome

⚠️ Commande pour utilisateur avancé. Utilisez plutôt le menu si vous n’êtes pas sûr de vous. Défini le biome du plot.

Informations

/plot help

Affiche le menu d’aide du plugin.

/plot confirm

Confirme une action lorsqu’une confirmation est demandée.

/plot info

Affiche des informations sur le plot sur lequel vous vous trouvez.

/plot list

Liste les plots. Vous pouvez utiliser différents arguments :

  • forsale : Plots à vendre.
  • mine : Plots que vous possédez.
  • shared : Plots sur lesquels vous êtes autorisé à construire.
  • top : Plots les plus votés.
  • all : Tous les plots.
  • joueur : Tous les plots d’un joueur.
  • done : Plots terminés.

/plot like

Met un like au plot sur lequel vous vous trouvez.

/plot unlike

Retire votre like d’un plot.

/plot target

Vise un plot avec votre compas.

Administration

/plot copy

Copie un plot aux coordonnées précisées

/plot move

Déplace un plot aux coordonnées précisées.

/plot swap

Echange deux plots de place.

/plot weanywhere

Autorise l’utilisation de WorldEdit partout sur le monde (même sur les routes et les plots d’autres joueurs).

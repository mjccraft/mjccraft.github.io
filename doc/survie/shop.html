Le Shop

Introduction :

Le serveur MJCCraft dispose d'un système monétaire. Celui-ci fonctionne de plusieurs façons. D'abord les joueurs peuvent marchander avec une banque virtuelle à fonds sans limites : Le serveur lui-même. Ainsi, vous pouvez acheter et vendre autant de choses que vous pouvez sans jamais être bloqué par celui-ci. Mais les prix sont réfléchis afin que les joueurs ne se réfèrent pas qu’à celui-ci. C’est pourquoi les joueurs peuvent également marchander entre eux afin de toucher des prix plus atteignables. Le système monétaire est construit de différentes manières qui sont les suivantes :

Le Marché

Le marché est une salle dans le "spawn" du monde survie aux coordonnées suivantes : X = 9506 ; Y = 72 ; Z = 1001514.

Ce marché vous permet de marchander avec le serveur. Le marché dispose de quatre salles à thèmes : Minéraux, Redstone, Divers et Blocks décoratifs. Ces salles ne marchandent pas tout les objets disponnibles du jeu : Une sélection réfléchie a été effectuée par les administrateurs afin que les joueurs ne se réfèrent pas uniquement à cette salle. Cette salle vous présente deux types de quantités possibles à aquérir ou à vendre : 1 objet à la fois ou par "stack" (64 objets / 16 objets (dépend de la quantité stockable par objet)). Mais certains objets ne peuvent être disponibles que par stocks de 1 (On ne peut pas les accumulés dans un même emplacement).

Le système d'achat et de vente fonctionnent à l'aide de panneaux :

  • Click droit pour acheter.
  • Click gauche pour vendre.

Votre économie

Vous pouvez créer votre propre marché dans votre base afin de vendre ce que vous désirez mais surtout pouvoir acheter à des prix plus bas des objets dont vous auriez besoin. Pour ce faire il vous suffit d'un panneau et d'un coffre. Nous vous conseillons fortement de "Claim" le chunk dans lequel vous placez votre marché. Pour commencer, placer un coffre (double ou simple peu importe). Puis placer un panneau sur un block adjacent au coffre. Sur le panneau il vous faut entrer avec la syntaxe suivante :

  • Première ligne : Ne rien entrer
  • Deuxième ligne : 1 --> La quantité
  • Troisième ligne : B 100 : S 50 --> Correspond au prix d'achat et S correspond au prix de vente
  • Quatrième ligne : ? --> Pour simplifier

Ensuite, prenez l'objet que vous voulez commercer et effectuez un click droit sur le panneau. Celui-ci devrait prendre l'apparence suivante :

  • Première ligne : "Votre Pseudo"
  • Deuxième ligne : 1
  • Troisième ligne : B 5000 : S 2000
  • Quatrième ligne : "Nom de l'objet"
